## Proyecto CodePen Clone — Curso CS100 (Lanbide)

Este proyecto es una réplica de la página principal de CodePen, desarrollada como práctica del curso CS100 (Introducción al Desarrollo Web) impartido por Lanbide.
El objetivo fue aplicar conceptos fundamentales de HTML y CSS para construir una interfaz moderna, limpia y responsiva.

## Tecnologías utilizadas

    •	HTML5 — estructura semántica del sitio
    •	CSS3 — estilos y diseño responsivo
    •	Google Fonts (Nunito) — tipografía principal
    •	Font Awesome — íconos vectoriales

## Características principales

    •	Cabecera con logo, navegación y buscador.
    •	Sección principal (hero section) con título, descripción, botones y llamada a la acción.
    •	Diseño inspirado en la interfaz oficial de CodePen.
    •	Uso de selectores y clases CSS para la maquetación y el estilo.

## Objetivo del ejercicio

Practicar:
• La correcta estructura de un documento HTML.
• La separación de contenido y estilo.
• El uso de fuentes externas e íconos.
• La creación de una interfaz coherente y visualmente atractiva.
